Bandplan.org - What is this?

Bandplan.org is an attempt to create an Open Source standard for ham radio band plans.

  • Only one central source for bandplan information
  • Worldwide coverage of bandplans
  • PDF, HTML bandplans can be created automatically
  • It is possible to create structures for further processing (i.e. in Arduino, any logging program)
  • Features foreign operations (license mappings, ...)
  • Comprehensive repeater database Follow the download links for frozen version with a (yet) limited set of functionality: * Latest release: # How to use it? ## Download and Install Down An example output can be found under html/ or the following link: [html bandplan](https://raw.github.com/8cH9azbsFifZ/bandplan/master/html/bandplan.html) #### Create the html output: From the main directory run: make html ### FT817 Controller using Arduino C structures for further use (i.e. in Arduino projects) can be created as well.
