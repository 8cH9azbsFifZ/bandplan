<XML>Bandplan.org - What is this?

<XML>Bandplan.org is an attempt to create an Open Source standard for ham radio band plans.

  • Only one central source for bandplan information
  • Worldwide coverage of bandplans
  • PDF, HTML bandplans can be created automatically
  • It is possible to create structures for further processing (i.e. in Arduino, any logging program)
  • Features foreign operations (license mappings, ...)
  • Comprehensive repeater database

How to use it?

Create the a HTML bandplan:

$ make html

C structures for further use (i.e. in Arduino projects) can be created as well. Here we create the headers containing bandplans and channels for an Yaesu FT-817: 

$ make arduino

Contribute Bandplans

As of today only the bandplans of Germany have been implemented. Other bandplans will follow soon: bandplan issues. Feel free to contact the author (g@ziegenhain.com) to find out how to contribute in this stage of development.
